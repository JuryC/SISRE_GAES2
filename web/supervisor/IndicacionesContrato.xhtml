Indicaciones de contrato
¡Importante! Seguir las siguientes indicaciones para crear un contrato.
  • Le recomendamos tener un borrador para realizar el registro de manera más rápida y segura.
  • Para poder adaptar el contrato a cada cliente por favor usar las convenciones mostradas en la lista.
    Ejemplo: Nos referimos al cliente NombreCliente con documento DocumentoCliente.
  • Indicar el orden del elemento añadido, para así poder organizarlo en el documento.
  • Elegir la cantidad exacta de elementos que contendrán las listas.
Convenciones (No usar espacios)
  • NombreCliente = Nombre y apellidos del cliente
  • DocumentoCliente = Número de documento del cliente
  • TipoDocCliente = Tipo de documento del cliente
  • CiudadInicialCliente = Ciudad desde donde se inicia el servicio
  • CiudadFinalCliente = Ciudad destino del servicio
  • ListaMateriales = Lista de materiales usados para el servicio
  • ValorDolares = Valor total calculado en dolares
  • ValorPesos = Valor total calculado en dolares
  • Fecha = Fecha actual
